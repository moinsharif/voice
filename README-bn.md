# Vask - ভয়েস-ভিত্তিক AI সঙ্গী প্রকল্প

## প্রকল্প পরিচয়

**Vask** একটি সম্পূর্ণ স্থানীয়ভাবে চালিত ভয়েস-ভিত্তিক AI সঙ্গী সিস্টেম যা ব্যক্তিগতকৃত, প্রসঙ্গ-সচেতন কথোপকথন প্রদান করে মেজাজ বিশ্লেষণ এবং ক্রমাগত শেখার সাথে। সিস্টেমটি সম্পূর্ণভাবে অফলাইনে চলে ওপেন-সোর্স AI মডেল ব্যবহার করে, বক্তৃতা স্বীকৃতি, মুখ সনাক্তকরণ, মেজাজ বিশ্লেষণ এবং কথোপকথনমূলক AI একীভূত করে অর্থপূর্ণ মিথস্ক্রিয়া তৈরি করতে যা দিনে দিনে শেখার মাধ্যমে উন্নত হয়।

## মূল বৈশিষ্ট্য

### 1. ভয়েস ইনপুট এবং স্পিচ-টু-টেক্সট রূপান্তর
- মাইক্রোফোন থেকে অডিও ক্যাপচার করুন
- স্পষ্ট বক্তৃতার জন্য ৮৫% নির্ভুলতা
- পটভূমি শব্দ ফিল্টারিং
- ৩ সেকেন্ডের মধ্যে স্বীকৃতি সম্পূর্ণ করুন
- একাধিক ভাষা সমর্থন

### 2. প্রসঙ্গ-সচেতন AI প্রতিক্রিয়া
- কথোপকথনের ইতিহাস বিবেচনা করে প্রতিক্রিয়া তৈরি করুন
- শেষ ১০টি বিনিময় সক্রিয় মেমরিতে রাখুন
- ব্যবহারকারীর মেজাজ অবস্থা বিবেচনা করুন
- ২ সেকেন্ডের মধ্যে প্রতিক্রিয়া তৈরি করুন
- সম্পূর্ণ অফলাইন AI মডেল ব্যবহার করুন

### 3. স্থায়ী কথোপকথন ইতিহাস
- সমস্ত কথোপকথন স্থানীয় সংরক্ষণাগারে সংরক্ষণ করুন
- সময়মুদ্রা, বার্তা, মেজাজ এবং মেটাডেটা সংরক্ষণ করুন
- ৫০০ms এর মধ্যে ঐতিহাসিক কথোপকথন পুনরুদ্ধার করুন
- তারিখ, মেজাজ বা কীওয়ার্ড দ্বারা অনুসন্ধান করুন
- নিরাপদ ডেটা মুছে ফেলার বিকল্প

### 4. মুখ সনাক্তকরণ এবং মুখের অভিব্যক্তি বিশ্লেষণ
- ওয়েবক্যাম অ্যাক্সেস করুন
- মুখের ল্যান্ডমার্ক এবং অভিব্যক্তি সনাক্ত করুন
- সুখ, দুঃখ, রাগ, অবাক এবং নিরপেক্ষ অবস্থা চিহ্নিত করুন
- প্রতি সেকেন্ডে ন্যূনতম ১৫ ফ্রেম প্রক্রিয়া করুন
- ভয়েস-শুধু মেজাজ সনাক্তকরণ সমর্থন করুন

### 5. মেজাজ সনাক্তকরণ এবং বিশ্লেষণ
- ভয়েস টোন এবং মুখের অভিব্যক্তি থেকে মেজাজ সনাক্ত করুন
- মেজাজ শ্রেণীবিভাগ: ইতিবাচক, নিরপেক্ষ, নেতিবাচক, মিশ্র
- মেজাজ পরিবর্তনের উপর ভিত্তি করে প্রতিক্রিয়া টোন সামঞ্জস্য করুন
- প্রতিটি সেশনের জন্য মেজাজ ইতিহাস বজায় রাখুন
- সেশন শেষে মেজাজ সারাংশ রিপোর্ট তৈরি করুন

### 6. দিনে দিনে শেখা এবং উন্নতি
- প্রতিদিন সমস্ত মিথস্ক্রিয়া বিশ্লেষণ করুন
- ব্যবহারকারীর পছন্দ এবং যোগাযোগ শৈলীতে প্যাটার্ন চিহ্নিত করুন
- শেখা পছন্দগুলি ব্যবহারকারী প্রোফাইলে আপডেট করুন
- ইতিবাচক প্রতিক্রিয়া পাওয়া প্রতিক্রিয়া ট্র্যাক করুন
- পূর্ববর্তী দিনের শেখা প্রয়োগ করুন

### 7. ব্যবহারকারী প্রোফাইল এবং ব্যক্তিগতকরণ
- প্রথম মিথস্ক্রিয়ায় ব্যবহারকারী প্রোফাইল তৈরি করুন
- যোগাযোগ শৈলী, পছন্দের বিষয় এবং মেজাজ প্যাটার্ন সংরক্ষণ করুন
- প্রতিক্রিয়া ব্যক্তিগতকরণের জন্য প্রোফাইল ব্যবহার করুন
- স্পষ্ট পছন্দ অবিলম্বে আপডেট করুন
- স্থানীয় এনক্রিপশন সহ সংরক্ষণ করুন

### 8. সম্পূর্ণ অফলাইন অপারেশন এবং ডেটা গোপনীয়তা
- ইন্টারনেট সংযোগ ছাড়াই চলুন
- সমস্ত প্রয়োজনীয় মডেল স্থানীয়ভাবে উপলব্ধ যাচাই করুন
- কোনো বাহ্যিক API কল নেই
- সমস্ত ব্যবহারকারী ডেটা স্থানীয়ভাবে সংরক্ষণ করুন
- কোনো ডেটা ট্রান্সমিশন নেই
- JSON/CSV ফর্ম্যাটে ডেটা এক্সপোর্ট করুন

### 9. টেক্সট-টু-স্পিচ আউটপুট
- AI প্রতিক্রিয়া অডিওতে রূপান্তরিত করুন
- সিস্টেম স্পিকারের মাধ্যমে প্লে করুন
- একাধিক ভয়েস এবং কথার হার সমর্থন করুন
- প্রতি ১০০ শব্দে ১ সেকেন্ডের মধ্যে সম্পূর্ণ করুন
- সম্পূর্ণ অফলাইন ভয়েস মডেল ব্যবহার করুন

### 10. সেশন ম্যানেজমেন্ট এবং রেকর্ডিং
- সেশন শুরু, বিরাম এবং শেষ করুন
- ওয়েক ওয়ার্ড বা বোতাম দিয়ে রেকর্ডিং শুরু করুন
- সেশন ডেটা কথোপকথন ইতিহাসে সংরক্ষণ করুন
- সেশন অডিও এবং ট্রান্সক্রিপ্ট পুনরায় চালান

### 11. মাল্টি-ব্যবহারকারী সমর্থন
- মুখ সনাক্তকরণ দ্বারা ব্যবহারকারী চিনুন
- প্রতিটি ব্যবহারকারীর জন্য আলাদা প্রোফাইল এবং ইতিহাস
- ম্যানুয়াল ব্যবহারকারী নির্বাচন বিকল্প
- ব্যবহারকারী ডেটা নিরাপদে আলাদা করুন

### 12. মেজাজ রিপোর্টিং এবং বিশ্লেষণ
- দৈনিক, সাপ্তাহিক এবং মাসিক মেজাজ পরিসংখ্যান
- মেজাজ প্রবণতা এবং প্যাটার্ন চিহ্নিত করুন
- কথোপকথনের বিষয় এবং সময়ের সাথে মেজাজ সম্পর্ক করুন
- চিন্তাজনক মেজাজ প্যাটার্নের জন্য সহায়ক পরামর্শ প্রদান করুন
- স্পষ্ট, ভিজ্যুয়াল ফর্ম্যাটে রিপোর্ট উপস্থাপন করুন



## প্রযুক্তি স্ট্যাক

### মূল উপাদান

| উপাদান | প্রযুক্তি | বিবরণ |
|--------|---------|--------|
| **ভয়েস স্বীকৃতি** | Whisper (OpenAI) / Vosk | অফলাইন স্পিচ-টু-টেক্সট |
| **AI মডেল** | Llama 2 / Mistral | স্থানীয় ভাষা মডেল |
| **মুখ সনাক্তকরণ** | OpenCV + MediaPipe | রিয়েল-টাইম মুখ এবং অভিব্যক্তি সনাক্তকরণ |
| **মেজাজ বিশ্লেষণ** | Custom ML Model | ভয়েস এবং মুখের অভিব্যক্তি থেকে মেজাজ সনাক্তকরণ |
| **টেক্সট-টু-স্পিচ** | Piper / Glow-TTS | অফলাইন ভয়েস সংশ্লেষণ |
| **ডেটা সংরক্ষণ** | SQLite / JSON | স্থানীয় ডেটাবেস এবং ফাইল সংরক্ষণ |
| **এনক্রিপশন** | AES-256 | সংবেদনশীল ডেটা এনক্রিপশন |
| **ফ্রেমওয়ার্ক** | Python / FastAPI | ব্যাকএন্ড এবং API |
| **ফ্রন্টএন্ড** | Electron / PyQt | ডেস্কটপ ইউজার ইন্টারফেস |

## ইনস্টলেশন এবং সেটআপ

### প্রয়োজনীয়তা

- **অপারেটিং সিস্টেম**: Windows, macOS, Linux
- **Python**: ৩.৯ বা উচ্চতর
- **RAM**: ন্যূনতম ৪GB (সুপারিশকৃত ৮GB+)
- **ডিস্ক স্পেস**: ২০GB+ (AI মডেলের জন্য)
- **ওয়েবক্যাম**: ঐচ্ছিক (মুখ সনাক্তকরণের জন্য)
- **মাইক্রোফোন**: প্রয়োজনীয় (ভয়েস ইনপুটের জন্য)

### ধাপ ১: রিপোজিটরি ক্লোন করুন

```bash
git clone https://github.com/yourusername/vask.git
cd vask
```

### ধাপ ২: ভার্চুয়াল পরিবেশ তৈরি করুন

```bash
python -m venv venv

# Windows এ
venv\Scripts\activate

# macOS/Linux এ
source venv/bin/activate
```

### ধাপ ৩: নির্ভরতা ইনস্টল করুন

```bash
pip install -r requirements.txt
```

### ধাপ ৪: AI মডেল ডাউনলোড করুন


## উন্নত বৈশিষ্ট্য

### দিনে দিনে শেখা

Vask প্রতিদিন আপনার মিথস্ক্রিয়া থেকে শিখে:
- আপনার যোগাযোগ শৈলী বিশ্লেষণ করে
- পছন্দের বিষয় চিহ্নিত করে
- মেজাজ প্যাটার্ন ট্র্যাক করে
- ইতিবাচক প্রতিক্রিয়া পাওয়া প্রতিক্রিয়া অগ্রাধিকার দেয়

এই শেখা স্বয়ংক্রিয়ভাবে প্রয়োগ করা হয় এবং আপনার অভিজ্ঞতা উন্নত করে।

### মাল্টি-ব্যবহারকারী সমর্থন

পরিবারের সদস্যরা তাদের নিজস্ব প্রোফাইল এবং কথোপকথন ইতিহাস সহ Vask ব্যবহার করতে পারে:
- মুখ স্বীকৃতি স্বয়ংক্রিয় ব্যবহারকারী সনাক্তকরণ
- ম্যানুয়াল ব্যবহারকারী নির্বাচন বিকল্প
- সম্পূর্ণ ডেটা বিচ্ছিন্নতা এবং গোপনীয়তা

### ডেটা এক্সপোর্ট

আপনার সমস্ত ডেটা এক্সপোর্ট করুন:
- কথোপকথন ইতিহাস (JSON/CSV)
- ব্যবহারকারী প্রোফাইল
- মেজাজ বিশ্লেষণ রিপোর্ট

সেটিংস > ডেটা > এক্সপোর্ট এ যান

```yaml
ai_model:

  enabled: true
  fps: 15

mood_analysis:
  enabled: true

# পারফরম্যান্স সেটিংস
performance:
  max_ram_usage: 2048  # MB
  enable_gpu: false
  num_threads: 4

privacy:
  encryption_enabled: true
```

## সমস্যা সমাধান

### সমস্যা: মাইক্রোফোন কাজ করছে না

**সমাধান:**
1. সিস্টেম সেটিংসে মাইক্রোফোন অনুমতি চেক করুন
2. অডিও ডিভাইস সঠিকভাবে সংযুক্ত আছে কিনা যাচাই করুন
3. অন্য অ্যাপ্লিকেশন মাইক্রোফোন ব্যবহার করছে কিনা চেক করুন

### সমস্যা: AI প্রতিক্রিয়া ধীর

**সমাধান:**
1. সিস্টেম সংস্থান পরীক্ষা করুন (RAM, CPU)
2. ছোট মডেল ব্যবহার করার চেষ্টা করুন
3. GPU ত্বরণ সক্ষম করুন (যদি উপলব্ধ হয়)

### সমস্যা: মুখ সনাক্তকরণ কাজ করছে না

**সমাধান:**
1. ওয়েবক্যাম অনুমতি চেক করুন
2. ওয়েবক্যাম সঠিকভাবে সংযুক্ত আছে কিনা যাচাই করুন
3. আলোর অবস্থা উন্নত করুন
4. ক্যামেরা লেন্স পরিষ্কার করুন
